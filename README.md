# iRemote

webrtc screensharing and remote control

## BUILD AND RUN FOR WINDOWS

### Prerequisites:

run as system admin
````
 npm install --global --production windows-build-tools
````

### RUN OR BUILD:

1. Clone this repo
````
git clone https://github.com/devarthurribeiro/iremote.git
````
2. Install modules 
````
yarn
````
3. Rebuild modules
````
yarn rebuild
````
4. run yarn start 
````
yarn start 
````
 or build
````
yarn dist 
````
